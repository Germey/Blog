在 OpenAI Codex CLI 里用 AceData Cloud MCP — 终端 AI 工具全能配置

作者 崔庆才 发表于 2026-05-27 分类于 产品推广 阅读次数： 本文字数： 3.2k 阅读时长 ≈ 3 分钟

这是一篇总览文章：把 AceData Cloud 提供的 11 个 MCP Server 一次性接到 Codex CLI，让 OpenAI 的开源终端编程工具在写代码之余顺手生图、写歌、做视频。

Codex CLI 是 OpenAI 官方推出的开源终端 AI 编程助手，原生支持 MCP。

AceData Cloud 提供哪些 MCP Server？

我们把以下能力封装成托管式远程 MCP Server，开箱即用：

Midjourney 🎨 绘画 文生图、混图、放大、Edit、生成视频 https://midjourney.mcp.acedata.cloud/mcp

一个 AceData Cloud Token = 全部 11 个 MCP Server，统一计费、按调用扣费，无月费。

为什么 Codex CLI + MCP

  • OpenAI 官方维护，长期更新有保障
  • 极简启动 codex，任何 shell 都能用
  • 通过 MCP 调用 AceData Cloud，绕过模型本身的能力上限（写歌、生图、视频）

第 1 步：获取 API Token

  1. 打开 AceData Cloud 平台，支持 GitHub、Google、微信、邮箱、手机号登录，注册即送免费额度。
  2. 登录后首页就能看到你的 API Token，点击复制。

  1. 余额不足时去 控制台 → 充值，支持支付宝、微信、Stripe。

计费与额度

  • 按次扣费，不调用不花钱，无月费、无最低消费。
  • 同一 Token 在所有 MCP Server 之间共享额度。
  • 所有调用记录、剩余额度、消费明细可在 控制台 查看。
  • 大用量需求可联系商务，享受批量价格。

常见问题

Q：可以只接入其中几个 MCP Server 吗？
A：当然，按需添加配置即可，没用到的就不加。

Q：有没有更便宜的接入方式？
A：直接调用 REST API 比 MCP 更便宜（少一层封装），但失去 MCP 在 LLM 工具调用上的便利。

Q：海外 / 跨境延迟问题？
A：所有 MCP Server 部署在香港，全球访问稳定；中国大陆直连无需代理。

Q：支持自托管吗？
A：MCP Server 全部 开源在 GitHub，可自行部署。
